A simple WP action to clear the RSS widget cashe.

== Description ==

Are you using the WP default RSS widget to display a feed on your personal page 
or blog? Did the damn thing cache a feed (or several feeds) and refreshing the 
page and browser cache doesn't work? Here is the solution...

Visit the [author's blog](http://milos.djekic.net) to stay in touch and don't 
miss [Synchi plugin](wordpress.org/extend/plugins/synchi/) - a full IDE inside 
your WordPress.

Install this simple plugin and run the Clear RSS Cache action from your Apperance 
menu whenever you want to fix this most anoying issue on your blog. Enjoy!

== Installation ==

If you are by chance using Internet Explorer 6, please do not install this plugin, 
even better - stop using Internet.

= Installation steps =

1. Upload 'wp-clear-rss-cashe' to the '/wp-content/plugins/' directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Use Apperance/Clear RSS Cache to perform the cleaning
1. That's it. Enjoy!
